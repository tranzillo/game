# Game (working title)

A browser-based, single-player, asymmetric, turn-based card game played against an AI opponent. Runs entirely in the browser as static HTML/JS/CSS — no backend, no install, accessible at a URL.

## Status

Pre-prototype. Currently capturing the concept and design in `DESIGN.md`. No code yet — framework choice deferred until after a vertical-slice prototype exists.

## Repo layout (so far)

- `README.md` — this file
- `DESIGN.md` — living design document; the source of truth for the game's concept, mechanics, and open questions
- `DECISIONS.md` — append-only log of decisions and the reasoning behind them
